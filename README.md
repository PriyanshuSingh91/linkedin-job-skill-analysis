# 📊 LinkedIn Job Skill Demand Analysis

Analysis of 4,700+ LinkedIn job listings to find the most in-demand 
data science skills in the USA.

## 🔧 Tools Used
- Python, Pandas, Matplotlib, Seaborn, Jupyter

## 📁 Files
- `linkedin_skill_demand_analysis.ipynb` — Main analysis notebook
- `linkedin-jobs-usa.csv` — Dataset

## 📌 Key Findings
- SQL and Python are the top 2 most demanded skills
- AWS, Tableau and Machine Learning follow closely
- California and New York lead in job listings
